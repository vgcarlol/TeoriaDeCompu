Ejercicio No. 2: Utilización del Lema de Arden para Encontrar el Lenguaje Generado por un Autómata Finito

Procedimiento

Paso 1: Identificación de Estados y Transiciones

El autómata finito dado tiene los siguientes estados y transiciones:

  • Estados: \( q_0, q_1, q_2, q_3, q_4, q_5, q_6, q_7, q_8, q_9, q_{10}, q_{11}, q_{12}, q_{13}, q_{14}, q_{15}, q_{16}, q_{17} \)

Paso 2: Escritura de Ecuaciones de Transición

Para cada estado \( q_i \), escribimos una ecuación que representa las transiciones hacia ese estado en términos de expresiones regulares:

  • \( q_0 = \epsilon q_1 + \epsilon q_2 + \epsilon q_3 \)
  • \( q_1 = 0 q_4 \)
  • \( q_2 = \epsilon q_4 \)
  • \( q_3 = \epsilon q_4 \)
  • \( q_4 = \epsilon q_5 \)
  • \( q_5 = \epsilon q_6 \)
  • \( q_6 = 1 q_7 + \epsilon q_8 \)
  • \( q_7 = \epsilon q_{11} \)
  • \( q_8 = \epsilon q_9 \)
  • \( q_9 = \epsilon q_{10} \)
  • \( q_{10} = \epsilon q_{11} \)
  • \( q_{11} = \epsilon q_{12} \)
  • \( q_{12} = \epsilon q_{14} \)
  • \( q_{13} = \epsilon q_{14} \)
  • \( q_{14} = \epsilon q_{15} + \epsilon q_{16} + \epsilon q_{13} + 0 q_{15} \)
  • \( q_{15} = \epsilon q_{16} + \epsilon q_{17} \)
  • \( q_{16} = 0 q_{14} \)
  • \( q_{17} = \epsilon \)

Paso 3: Resolución de las Ecuaciones usando el Lema de Arden

  1. Para \( q_{17} \):
    \( q_{17} = \epsilon \)
  2. Para \( q_{16} \):
    \( q_{16} = 0 q_{14} q_{16} + \epsilon q_{17} \)
    Aplicando el Lema de Arden:
    \( q_{16} = (0 q_{14})^* \epsilon \)
  3. Para \( q_{15} \):
    \( q_{15} = \epsilon q_{16} + \epsilon q_{17} \)
    \( q_{15} = \epsilon (0 q_{14})^* \epsilon + \epsilon \)
    \( q_{15} = \epsilon (0 q_{14})^* \epsilon \)
  4. Para \( q_{14} \):
    \( q_{14} = \epsilon q_{15} + \epsilon q_{16} + \epsilon q_{13} + 0 q_{15} \)
    \( q_{14} = \epsilon (\epsilon (0 q_{14})^* \epsilon + \epsilon) + \epsilon (0 q_{14})^* \epsilon + \epsilon q_{13} + 0 (\epsilon (0 q_{14})^* \epsilon + \epsilon) \)
    Simplificando:
    \( q_{14} = \epsilon^2 (0 q_{14})^* \epsilon + \epsilon^2 + \epsilon (0 q_{14})^* \epsilon + \epsilon q_{13} + 0 \epsilon (0 q_{14})^* \epsilon + 0 \epsilon \)
    \( q_{14} = \epsilon^2 (0 q_{14})^* \epsilon + \epsilon^2 + \epsilon (0 q_{14})^* \epsilon + \epsilon q_{13} \)

Paso 4: Sustituciones Iterativas

  • Para \( q_{12} \) y \( q_{13} \):
    \( q_{12} = \epsilon q_{14} \)
    \( q_{13} = \epsilon q_{14} \)
  • Para \( q_{11} \):
    \( q_{11} = \epsilon q_{12} \)
    \( q_{11} = \epsilon (\epsilon q_{14}) \)
    \( q_{11} = \epsilon^2 q_{14} \)
  • Para \( q_{10} \):
    \( q_{10} = \epsilon q_{11} \)
    \( q_{10} = \epsilon (\epsilon^2 q_{14}) \)
    \( q_{10} = \epsilon^3 q_{14} \)
  • Para \( q_{9} \):
    \( q_{9} = \epsilon q_{10} \)
    \( q_{9} = \epsilon (\epsilon^3 q_{14}) \)
    \( q_{9} = \epsilon^4 q_{14} \)
  • Para \( q_{8} \):
    \( q_{8} = \epsilon q_{9} \)
    \( q_{8} = \epsilon (\epsilon^4 q_{14}) \)
    \( q_{8} = \epsilon^5 q_{14} \)
  • Para \( q_{7} \):
    \( q_{7} = \epsilon q_{11} \)
    \( q_{7} = \epsilon (\epsilon^2 q_{14}) \)
    \( q_{7} = \epsilon^3 q_{14} \)
  • Para \( q_{6} \):
    \( q_{6} = 1 q_{7} + \epsilon q_{8} \)
    \( q_{6} = 1 (\epsilon^3 q_{14}) + \epsilon (\epsilon^5 q_{14}) \)
    \( q_{6} = \epsilon^3 q_{14} + \epsilon^6 q_{14} \)
  • Para \( q_{5} \):
    \( q_{5} = \epsilon q_{6} \)
    \( q_{5} = \epsilon (\epsilon^3 q_{14} + \epsilon^6 q_{14}) \)
    \( q_{5} = \epsilon^4 q_{14} + \epsilon^7 q_{14} \)
  • Para \( q_{4} \):
    \( q_{4} = \epsilon q_{5} \)
    \( q_{4} = \epsilon (\epsilon^4 q_{14} + \epsilon^7 q_{14}) \)
    \( q_{4} = \epsilon^5 q_{14} + \epsilon^8 q_{14} \)
  • Para \( q_{3} \):
    \( q_{3} = \epsilon q_{4} \)
    \( q_{3} = \epsilon (\epsilon^5 q_{14} + \epsilon^8 q_{14}) \)
    \( q_{3} = \epsilon^6 q_{14} + \epsilon^9 q_{14} \)
  • Para \( q_{2} \):
    \( q_{2} = \epsilon q_{4} \)
    \( q_{2} = \epsilon (\epsilon^5 q_{14} + \epsilon^8 q_{14}) \)
    \( q_{2} = \epsilon^6 q_{14} + \epsilon^9 q_{14} \)
  • Para \( q_{1} \):
    \( q_{1} = 0 q_{4} \)
    \( q_{1} = 0 (\epsilon^5 q_{14} + \epsilon^8 q_{14}) \)
    \( q_{1} = 0 \epsilon^5 q_{14} + 0 \epsilon^8 q_{14} \)
  • Para \( q_{0} \):
    \( q_{0} = \epsilon q_{1} + \epsilon q_{2} + \epsilon q_{3} \)
    \( q_{0} = \epsilon (0 \epsilon^5 q_{14} + 0 \epsilon^8 q_{14}) + \epsilon (ε^6 q_{14} + \epsilon^9 q_{14}) + \epsilon (\epsilon^6 q_{14} + \epsilon^9 q_{14}) \)
    Simplificando finalmente:
    \( q_{0} = \epsilon (0 \epsilon^5 q_{14} + 0 \epsilon^8 q_{14}) + \epsilon (2 \epsilon^6 q_{14} + 2 \epsilon^9 q_{14}) \)

Expresión Regular Final

\( q_{0} = \epsilon (0 \epsilon^5 q_{14} + 0 \epsilon^8 q_{14}) + \epsilon (2 \epsilon^6 q_{14} + 2 \epsilon^9 q_{14}) \)

Conclusión

El procedimiento para encontrar la expresión regular utilizando el Lema de Arden implica escribir las ecuaciones de transición para cada estado, resolverlas iterativamente de los estados finales hacia los estados iniciales y simplificar las expresiones resultantes.
